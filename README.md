# Sistema de notas em C

Mini-projeto desenvolvido em C com o objetivo de praticar o uso de matrizes, funções e laços de repetição.

O programa lê as notas de alguns alunos, armazena os valores em uma matriz e calcula a média individual, a média da turma, além de identificar a maior e a menor média.

## Funcionalidades
- Leitura das notas dos alunos
- Cálculo da média de cada aluno
- Cálculo da média da turma
- Identificação da maior e da menor média

## Objetivo do projeto
Treinar lógica de programação e consolidar conceitos básicos da linguagem C, principalmente o uso de matrizes, funções e estruturas de repetição.

## Como executar
Compile e execute o arquivo principal:

```bash
gcc sistema_de_notas.c -o sistema_de_notas
./sistema_de_notas
```
